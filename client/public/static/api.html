POST /api/dictionary/create

空の辞書を新しく作成します。

リクエストボディ

{
name: string
辞書の名称
}

レスポンス

Dictionary
作成した辞書

401-

POST /api/dictionary/upload

ファイルをアップロードし、現在のデータを上書きします。 OTM-JSON 形式 (拡張子 .json) と PDIC 形式 (拡張子 .dic) に対応しています。

ファイルのアップロードが完了した段階でレスポンスが返されます。 ファイルのパース処理やデータベースの更新処理は、レスポンスが返された後にバックグラウンドで行われます。

リクエストボディ

{
number: number
辞書の ID 番号
}

レスポンス

Dictionary
作成した辞書

エラー

400noSuchDictionaryNumber
与えられた ID 番号の辞書は存在しません。
401-
ユーザー情報が不正です。
403-
該当辞書にアクセスする権限がありません。
